Portfolio

Personal portfolio - the site you're looking at right now!

  • Frontend is a static HTML site
  • Backend uses FastAPI to act as a data aggregator and implement small features like playing music, GitHub/Leetcode widgets (coming soon)
  • HTMX, JS for dynamic components

This may seem unnecessarily complicated; it's actually my solution to a legitimate problem. I’m hosting on Render’s free tier; for a full application (anything more complicated than a static site) Render takes up to a minute to spin up an instance after inactivity. I wanted my site to showcase my personality and have interesting components, but I also wanted the main page to be fast-loading. This approach lets me have the best of both worlds.
